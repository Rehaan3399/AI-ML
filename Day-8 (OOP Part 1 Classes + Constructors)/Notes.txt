# What is OOP?

OOP (Object-Oriented Programming) is a way of organizing code using classes and objects.

Instead of storing data and functions separately, OOP keeps them together.

Example:

Bank Account:
- account holder name
- balance
- deposit()
- withdraw()

All belong to one object.
---
# Class
A class is a blueprint/template for creating objects.
Example:
```python
class Student:
    pass
```

Class itself is not a real student.

It is only a design.

---
# Object

An object is an actual instance created from a class.

Example:

```python
s1 = Student()
```

Here:

```text
Student → Class
s1 → Object
```

---

# self Keyword

self refers to the current object.

Example:

```python
class Student:

    def greet(self):
        print("Hello")
```

When:

```python
s1.greet()
```

Python internally does:

```python
Student.greet(s1)
```

Therefore self refers to s1.

---

# Constructor (__init__)

Constructor runs automatically when object is created.

Example:

```python
class Student:

    def __init__(self,name):
        self.name = name
```

Object creation:

```python
s1 = Student("Mohammad")
```

Python automatically calls:

```python
__init__()
```

---

# Instance Variables

Variables belonging to each object.

Example:

```python
self.name = name
self.age = age
```

Each object stores its own values.

Example:

```python
s1 = Student("Mohammad")
s2 = Student("Rahul")
```

Output:

```text
s1.name → Mohammad
s2.name → Rahul
```

Different values for different objects.

---

# Class Variables

Variables shared by all objects.

Example:

```python
class Student:

    school = "ABC College"
```

All objects access same value.

Example:

```python
s1.school
s2.school
```

Both return:

```text
ABC College
```

---

# Instance Methods

Functions inside class.

Example:

```python
class Calculator:

    def add(self,a,b):
        return a+b
```

Call:

```python
c1 = Calculator()

c1.add(10,20)
```

Output:

```text
30
```

# Important Differences

| Concept | Meaning |
|----------|----------|
| Class | Blueprint |
| Object | Real instance |
| self | Current object |
| __init__ | Constructor |
| Instance Variable | Belongs to object |
| Class Variable | Shared by class |
| Method | Function inside class |

---

# Common Beginner Mistakes

## Forgetting self

Wrong:

```python
def deposit(amount):
```

Correct:

```python
def deposit(self,amount):
```

---

## Forgetting object creation

Wrong:

```python
deposit(500)
```

Correct:

```python
account.deposit(500)
```

---

## Confusing class and object

Wrong thinking:

```text
Class = Object
```

Correct:

```text
Class → Blueprint
Object → Actual instance
```
