# PYTHON: “EVERYTHING IS OBJECT” — FULL CLARITY NOTE
## 1. Core Meaning
> Python lo almost prati value ni object laga treat chestaru.

Object ante:

* Data untundi
* Functions/abilities untayi
* Memory lo store avutundi
* Class nundi create avutundi
---

# 2. Real World Understanding
## Example: Mobile Phone
### Data (Properties)

* color
* battery
* brand

### Actions (Functions)

* call()
* charge()
* play_music()
So:
```text
Object = Data + Actions
```Same concept Python lo use chestaru.
---

# 3. Python Examples
## String Object
```python id="btjlwm"
name = "Mohammad"
```
`"Mohammad"` is a STRING OBJECT.
It has:
* data → Mohammad
* functions → upper(), lower(), replace()
Example:
```python id="pnodjlwm"
print(name.upper())
```
Output:
```python id="tjlwm"
MOHAMMAD
```
Meaning:
String object ki own abilities untayi.
---

# 4. List Object
```python id="jlwmx1"
marks = [90, 80, 70]
```
List object has functions:
```python id="jlwmx2"
marks.append(100)
```
Output:
```python id="jlwmx3"
[90, 80, 70, 100]
```
Meaning:
List also behaves like object.
---

# 5. Even Numbers are Objects
```python id="jlwmx4"
x = 10
```
Check type:
```python id="jlwmx5"
print(type(x))
```
Output:
```python id="jlwmx6"
<class 'int'>
```
Meaning:
10 is object created from `int` class.
---

# 6. Important Understanding
Most beginners think:
```python id="jlwmx7"
x = 10
```
means:

```text id="jlwmx8"
x directly stores 10
```
Not fully correct.

Actual concept:

```text id="jlwmx9"
Python creates INTEGER OBJECT in memory
x refers to that object
```
---

# 8. Why Python Says “Everything is Object”

Because every value has:

* type
* memory
* functions
* behavior

Examples:

| Value     | Object Type    |
| --------- | -------------- |
| `"hello"` | string object  |
| `10`      | integer object |
| `[1,2]`   | list object    |
| `3.5`     | float object   |

---

# 9. Easy Robot Analogy

Imagine every value is small robot.

## String Robot

* upper()
* lower()

## List Robot

Can:

* append()
* remove()

## Integer Robot

Can:

* do math

Each robot:

* has data
* has abilities

That robot = OBJECT.
